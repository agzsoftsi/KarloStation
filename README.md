## Description

Avatar generator App is a Simple tool to create custom avatars with at least 500 kinds of combinations using the power of Web API Canva, only create, download and share.


## Features: 

- Genders (2)
- Backgrounds (10)
- Avatar Name
- Foreground (10)
- Skin Colors(6)
- Eyesbrows (6) - Color (10)
- Eyes(13) - Color(10)
- Mouth(13)
- Hair(28) - Color(10)
- Beard(10)- Color(10)
- Accessories (10)
- Shirts(11) - Color(9)
- Prints (40)
- Pants(10) - Color(9)
- Shoes(6) - Color(6)
